# Project Rules

## Shell Commands

- **NEVER `cd` to the project directory.** The working directory is ALWAYS already correct. Run commands directly without any `cd` prefix.
- DO NOT compound expressions with `cd /c/Users/steve/code/heb &&` or any variant. Just run the command.


## Git and Deploy

- DO NOT COMMIT or PUSH to Git unless specifically requested to do so/
- DO NOT DEPLOY to itch.io or build game deployment files unless specifially requested to do so.
